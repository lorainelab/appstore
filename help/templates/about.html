Integrated Genome Browser and IGB Apps

Integrated Genome Browser (IGB) is a highly configurable, interactive and fast open source desktop genome browser. IGB provides hundreds of features, from opening and viewing data files to performing visual analytics functions on loaded data.

Now, IGB is also a platform for developers to experiment with new ways of visualizating genomic data. Inspired by the Cytoscape project, we developed this site to help developers distribute new IGB Apps and to help users find Apps that will help them in their work.

If you have ideas about how we improve this site, please let us know!
